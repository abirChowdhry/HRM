# HRM Database Export

This folder contains the SQL Server database export for the Assignment 2 submission.

## File

- `HRM_database_export_schema.sql`

## Notes

This export is schema-only. It creates the database structure from the Entity Framework Core migrations and does not include seed/sample records.

To use it, create or select the `HRM` database in SQL Server Management Studio 22, open the SQL file, and execute the script.
